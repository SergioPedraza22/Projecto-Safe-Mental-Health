                situación o pensamiento que lo haga sentir a uno frustrado, furioso o nervioso.
                El estrés es la reacción de su cuerpo a un desafío o demanda. En pequeños
                brotes, el estrés puede ser positivo, como cuando lo ayuda a evitar el peligro o
                cumplir con una fecha límite. Pero cuando el estrés dura mucho tiempo, puede
                dañar su salud.
            </p>
            <p style="font-size: 1.1em; line-height: 1.7; color: #333; text-align: justify; padding: 0 15px;">

                <br>- Problemas para dormir
                <br>- Cansancio o fatiga
                <br>- Dolores musculares
                <br>- Resfriados a repetición
                <br>- Diarrea o estreñimiento
